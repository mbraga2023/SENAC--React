App de contatos
Gerenciar uma lista de contatos pessoais.
Endpoint da API: [em breve]
Nome AppContatos
Campos: 
- id
- Nome
- Whatsapp
- E-mail

Funcionalidades: 
- Listar os contatos em ordem alfabética "com separador de letras"
- Opção para remover um conta da lista
	- solicitar uma confirmação de exclusão
- Opção para alterar dados de um contato, exceto o id.
- Formulário para inclusão dos campos acima.
	- máscara de entrada para o Whatsapp
	- validação para os campos email e Whatsapp (validar formato)
